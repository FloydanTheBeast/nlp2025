# Домашнее задание: Классификация текста на AG News

# Комментарии по решению

Со стандартный подходом с использованием TextCNN + BiLSTM не смог добиться F1 > 93%, так что решил использовать трансформеры

Финальное решение в файле [ag-news-classifier-transformers.ipynb](./ag-news-classifier-transformers.ipynb)

## Задание

Необходимо создать ноутбук и добиться качества **95% F1-score (macro)** на датасете AG News.

## Требования

- Датасет: AG News
- Метрика: Macro F1-score ≥ 0.95 на test наборе
- Можно использовать любые подходы из семинара (TextCNN, BiLSTM, предобученные эмбеддинги)
- Можно экспериментировать с архитектурой, гиперпараметрами, техниками регуляризации

## Рекомендации

- Используйте предобученные эмбеддинги (Word2Vec, GloVe, FastText)
- Экспериментируйте с архитектурой модели
- Применяйте техники регуляризации (dropout, batch normalization)
- Используйте learning rate scheduling
- Нельзя использовать ансамбли моделей

## Отчёт

В ноутбуке должны быть:

- Визуализация обучения (loss, accuracy, F1)
- Confusion matrix на test наборе
